# List API for Development

This is API for GET User Profile
```
get: https://library-crud-sample.vercel.app/api/user/profile
```

This is API for POST User Login
```dotnetcli
https://library-crud-sample.vercel.app/api/user/login
```

Body: JSON:
```json
{
  "email": "abimanyu945@gmail.com",
  "password": "password"
}
```
This us API for User Register
```
https://library-crud-sample.vercel.app/api/user/register
```

Body: JSON:
```json
{
  "name": "abi",
  "email": "abimanyu945@gmail.com",
  "password": "password"
}
```


## How to run
Please type | npm run dev for running development mode
